## 快速评分参考表

### 吸量评估标准
| 分数 | 标准 |
|------|------|
| 5 | 题材热门、视觉冲击强、适合短视频传播、话题性强 |
| 4 | 有一定吸引力、目标受众明确、传播潜力中等 |
| 3 | 小众题材、需要教育市场、传播难度较大 |
| 2 | 非常小众、商业价值有限 |
| 1 | 几乎无商业吸引力 |

### 技术难度标准
| 分数 | 标准 |
|------|------|
| 5 | 需要创新技术、开发风险高、专业团队才能完成 |
| 4 | 技术复杂度中高、需要特定领域专家 |
| 3 | 技术成熟但实现工作量中等 |
| 2 | 技术简单、成熟方案可直接使用 |
| 1 | 极简实现、个人开发者可完成 |

### 运营难度标准
| 分数 | 标准 |
|------|------|
| 5 | 内容消耗快、需要持续高投入、专业运营团队 |
| 4 | 内容需求中等、需要定期更新 |
| 3 | 内容消耗慢、更新压力小 |
| 2 | 易于扩展、玩家创作内容可持续 |
| 1 | 几乎无需运营、一次性内容即可 |

### 移动端适配性标准
| 分数 | 标准 |
|------|------|
| 5 | 触屏操作天然适配、碎片时间友好、性能要求低 |
| 4 | 适配手机但需要优化、操作稍复杂 |
| 3 | 可移植但体验打折、界面需要大幅调整 |
| 2 | 移动端体验差、不建议移植 |
| 1 | 完全不适合移动端 |
