# game-idea-collector Skill

## 技能描述
自动收集、分析市场上有趣的游戏创意玩法或设计，生成可复用的游戏创意分析报告。

## 适用场景
- 游戏策划需要灵感来源
- 新项目立项前的市场调研
- 寻找可复用的创新玩法机制
- 竞品分析和趋势研究

## 功能特性

### 1. 游戏创意采集
- 自动搜索独立游戏、热门手游、创新玩法
- 覆盖 PC/主机/手机/网页全平台
- 采集来源：Steam、indienova、Gcores、游戏媒体等

### 2. 多维度分析
每个游戏创意包含以下分析维度：
- **目标受众分析**: 用户画像、市场规模
- **玩法特色与核心机制**: 创新点、差异化
- **商业化潜力**: 变现模式、付费点设计
- **技术实现难度**: 开发周期、人力成本评估
- **移动端适配性**: 触屏适配、碎片化适配
- **运营推广可行性**: 传播性、直播友好度

### 3. 筛选标准
- 玩法创意/独特性 ⭐⭐⭐⭐⭐
- 用户吸引力 (吸量) ⭐⭐⭐⭐⭐
- 运营推广便利性 ⭐⭐⭐⭐
- 移动端适配性 ⭐⭐⭐⭐⭐

### 4. 输出格式
生成结构化 Markdown 报告，包含：
- 报告摘要 (TOP3 推荐)
- 详细分析表格
- 趋势分析与建议
- 技术实现参考
- 移动端适配优先级

## 使用方法

### 基础调用
```
请帮我收集分析最新的游戏创意玩法，生成分析报告
```

### 定向采集
```
收集 Roguelike 品类的创新玩法创意
收集适合移动端的休闲游戏创意
收集具有病毒传播潜力的游戏机制
```

### 输出配置
```
生成游戏创意报告，重点关注：
- 收集数量：10-15 款
- 分析维度：商业化潜力、技术难度
- 输出格式：Markdown 报告 + Excel 表格
```

## 工作流程

1. **搜索采集**
   - web_search 搜索游戏创意关键词
   - web_fetch 获取详细游戏介绍和评测

2. **整理分析**
   - 提取核心玩法机制
   - 评估各维度得分
   - 计算综合评分

3. **报告生成**
   - 结构化 Markdown 报告
   - 包含详细分析表格
   - 提供可复用设计建议

4. **文档提交**
   - 保存到 AIPlans GitHub 仓库
   - 飞书发送总结汇报

## 评分标准

### 吸量评估
- ⭐⭐⭐⭐⭐: 极强 (IP 加持/话题性强/病毒传播)
- ⭐⭐⭐⭐: 强 (玩法新颖/受众广泛)
- ⭐⭐⭐: 中 (稳定受众/差异化一般)
- ⭐⭐: 弱 (小众题材/竞争激烈)

### 移动端适配性
- ⭐⭐⭐⭐⭐: 完美适配 (触屏友好/碎片化)
- ⭐⭐⭐⭐: 良好适配 (需少量优化)
- ⭐⭐⭐: 可适配 (需中等改造)
- ⭐⭐: 较难适配 (操作复杂/时长要求高)

### 技术实现难度
- ⭐: 极低 (1 人月内)
- ⭐⭐: 低 (1-2 人月)
- ⭐⭐⭐: 中 (3-4 人月)
- ⭐⭐⭐⭐: 高 (5-8 人月)
- ⭐⭐⭐⭐⭐: 极高 (8 人月+)

## 注意事项

1. **数据来源**: 优先选择权威游戏媒体和官方渠道
2. **版权合规**: 报告仅供内部参考，避免直接复制玩法
3. **时效性**: 标注报告生成日期，定期更新
4. **本地化**: 考虑国内市场和用户偏好
